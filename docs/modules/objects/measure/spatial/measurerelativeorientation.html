Modules > Objects > Measure > Spatial

Measure relative orientation

Measures the orientation of each object relative to a specific point.

Description

Measures the orientation of each object relative to a specific point. Orientation of the objects themselves is provided by a previously-calculated measurement associated with each object. The relative orientation is the angle between the vector specified by this measurement and a vector passing from the object centroid to a specific point.

Note: Currently only works for X-Y plane measurements

Parameters

Parameter Description
Input objectsObjects from the workspace for which orientation relative to a point will be measured. Orientation of the objects themselves is provided by a previously-calculated measurement associated with each object. The relative orientation is the angle between the vector specified by this measurement and a vector passing from the object centroid to a specific point. Output relative orientation measurements are associated with the relevant input object.
Orientation in X/Y measurementMeasurement associated with each input object which defines the orientation of the first vector for the relative orientation calculation. The other vector passes between the object centroid and a specific point (controlled by the other parameters). This orientation measurement is assumed to be specified in degree units.
Measurement rangeThe range over which output relative orientation measurements are specified. For cases where the orientation of the object could be inverted without consequence (e.g. fitting orientation to an ellipse), the range "0-90 degrees" can be used; however, in cases where the orientation is more specific (e.g. having come from object tracking), the range "0-180 degrees" can be used. Relative orientation measurements are absolute, in that there's no distinction between positive and negative angular differences.
Reference modeControls the source of the reference point to which object orientations are compared. :
  • "Object centroid to image centre" Object orientations are compared to the centre of the object collection (i.e. to the image from which they were first detected).
  • "Object centroid to target centroid" Object orientations are compared to the centre of a specific object. Since only a single object can be used per timepoint, this will be either the smallest or largest object (controlled by "Object choice mode") in the specified collection ("Reference objects"). If "Reference must be in same frame" is not selected, the same object will be used in all timepoints.
  • "Object centroid to target surface" Object orientations are compared to the closest point on the surface of a specific object. In this mode, the reference points can be different for all objects. Since only a single object can be used per timepoint, this will be either the smallest or largest object (controlled by "Object choice mode") in the specified collection ("Reference objects"). If "Reference must be in same frame" is not selected, the same object will be used in all timepoints.
Reference objectsIf "Reference mode" is set to "Object centroid to target centroid" or "Object centroid to target surface", this is the object collection from which the reference object(s) will be drawn.
Object choice modeIf "Reference mode" is set to "Object centroid to target centroid" or "Object centroid to target surface", this controls whether the smallest or largest object from that collection will be used as the reference point for relative orientation calculations.
Reference must be in same frameWhen selected, the reference objects must be in the same frame as the input object being measured. As such, all the objects in one frame will be compared to a common reference, but won't necessarily have the same reference as an object in another frame.
